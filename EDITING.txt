= Editing of Asciidoctor Cheatsheets

== General

* Run `asciidoctor -a toc=left FILE.adoc`
* Table format:

    [cols="m,d"]
    |===

* Start with `:toc:`

== Replace

* `^([ |] )([A-Za-z0-9_]+\.)+([A-Za-z_]+)([ (])` -> `\1\2*\3*\4`
* `(\|.{43})\|` --> `\1|` (modify as needed)
* `^\n\|===` -> `\n[col="m,d"]\n|===`
