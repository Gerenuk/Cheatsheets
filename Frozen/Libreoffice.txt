= Librecalc data loading

Librecalc zu Mysql:
mysql-jdbc installieren
Libreoffice Extras/Optionen/Erweitert/Classpath auf C:\Program Files (x86)\MySQL\MySQL Connector J\mysql-connector-java-5.1.34-bin.jar zeigen lassen

Table from CSV:
* copy from Librecalc
* open Librebase; select Tables; Ctrl-V
* if you autodetect types you still might need to increase VARCHAR values!
